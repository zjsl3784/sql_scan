<도구 개요>  
SQL Injection 반응 및 포인트를 찾는 도구입니다. (sqlmap 처럼 데이터 추출용 아님)

   　  
<세부로직>  
사용자가 선택한 Request 값의 파라미터에 sql injection 반응을 테스트함. (GET, POST, JSON 형식 가능)  
연산자, 문자연결 등을 통한 블라인드 및 에러기반 검사를 수행함.  
time기반 로직은 없음. union, 스택쿼리 등 데이터 추출에 관련된 로직도 없음.    
사용자가 입력한 요청으로 content lenght 3회 측정해서 기본 base를 가져감.
블라인드 기반 검사가 주이기에 정상 결과가 존재하는 파라미터 값을 넣어야함. (대충 아무값이나 넣으면 의미가없음)  
값이 없는 파라미터는 검사에서 제외되며, 여러개의 파라미터가 존재할 경우 각 파라미터 별로 하나씩 테스트를 수행함.
